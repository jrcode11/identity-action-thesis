# Identity in Action: How Self-Perception Shapes Moral Behaviour

Bachelor thesis analysis examining how identity-related beliefs and perspective-taking influence moral decisions in an interactive narrative game.

**Author:** Jonas Retz  
**Institution:** Tilburg University, School of Humanities and Digital Sciences  
**Programme:** BSc Cognitive Science and Artificial Intelligence  
**Supervisor:** Dr. Silvy Collin  

## Overview

This repository contains all materials supporting the bachelor thesis, including the full analysis notebook, generated figures, and the survey instrument administered to participants.

The study employed a between-subjects experimental design (N = 85) testing three hypotheses about whether perspective-taking framing during interactive gameplay shifts moral decision-making, identity polarisation, and alignment between identity and choice.

## Repository Contents

- `analysis_final.ipynb` — Full Python notebook with data processing, statistical tests, and figure generation
- `figures/` — Four high-resolution figures used in the thesis Results section
- `materials/Bachelor_Questionnaire.pdf` — Complete survey instrument used in the study
- `requirements.txt` — Python library versions required to reproduce the analysis

## Reproducibility

The analysis was conducted in Python 3 using:

- pandas 2.3.3
- NumPy 2.3.5
- SciPy 1.16.3
- matplotlib 3.10.6

To reproduce the analysis, install the required libraries via:

​```
pip install -r requirements.txt
​```

Then open `analysis_final.ipynb` in Jupyter and run all cells in order.

Raw participant data is not included in this repository to protect participant confidentiality. The notebook documents all data corrections, exclusions, and analytic decisions.
